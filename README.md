# job Dating framework

A lightweight PHP MVC framework for building job dating applications.

## Features

- MVC architecture
- Secure database interactions with prepared statements
- Input validation
- User authentication
- Easy routing system
- Templating engine for views

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yassir-lazreq/job_dating.git
   ```

## Requirements

- PHP 7.4 or higher
- MySQL or compatible database
- Composer

## Usage

1. Configure your database settings in `config/database.php`.
2. Set up your web server to point to the `public` directory.
3. Create your models, views, and controllers following the MVC pattern.
4. Use the routing system to define your application routes in `app/routes.php`.
5. Start building your job dating application!
